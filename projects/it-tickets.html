Automated IT Support Ticket Intelligence System

Process IT support tickets using NLP, visualize priorities, and detect patterns in high-priority requests.

Technologies Used:

Live ML API

This project includes a deployed ML API that predicts whether and IT support ticket is high priority. The model is served through a REST API built with FastAPI and deployed to the cloud.
